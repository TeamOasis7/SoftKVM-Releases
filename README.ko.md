# Soft KVM

[English](README.md) | **한국어**

Soft KVM은 같은 로컬 네트워크의 Windows PC를 하나의 마우스와 키보드로 제어하는 프로그램입니다.

## 빠른 시작

1. 제어받을 PC에서 **Wait**를 누르고 2자리 코드를 확인합니다.
2. 제어할 PC에서 **Control**을 누르고 해당 코드를 입력합니다.
3. **Start**를 눌러 연결합니다.
4. 사용을 마치면 **Stop**을 눌러 연결을 종료합니다.

코드 입력칸 옆의 `×`를 누르면 연결하지 않고 취소합니다. 연결은 유지하면서 입력 전달만 잠시 멈추려면 **Pause**를 사용합니다.

## 명령창

`K`와 `M`을 1초간 함께 누른 뒤 두 키를 놓습니다. 이어서 다음 키를 누릅니다.

- `X`: 제어 PC의 로컬 클릭 차단 켜기/끄기
- `K`: 제어 PC의 로컬 키보드 차단 켜기/끄기
- `G`: 게임 입력 모드 켜기/끄기
- `Z`: 원격 입력 일시정지/재개
- `E`: 즉시 중단하고 입력 복구
- `Esc`: 명령창 닫기

명령창이 닫혀 있을 때 일반 `Alt` 단축키와 `Esc`는 원격 PC로 전달됩니다.

## Settings

- **Start with Windows**: Windows 로그인 후 Soft KVM을 자동으로 시작합니다. 창은 알림 영역에 숨겨지며 트레이 아이콘을 더블클릭하면 열립니다.
- **Minimize while waiting**: 연결된 뒤 대기 PC의 창을 숨깁니다.
- **Keyboard control**: 키보드 입력을 대기 PC로 전달합니다.
- **Mouse control**: 마우스 입력을 대기 PC로 전달합니다.
- **Block local key**: 원격 키 입력은 계속 보내면서 제어 PC의 로컬 입력을 막습니다.
- **Block local clicks**: 원격 클릭은 계속 보내면서 제어 PC의 Soft KVM 창 바깥 클릭을 막습니다.
- **Game input mode**: 게임과 3D 프로그램에 필요한 상대 마우스 이동을 사용합니다.
- **Connection group**: 같은 그룹 이름을 사용하는 PC만 검색합니다.
- **Control UAC screens**: 관리자 승인 후 UAC 화면과 관리자 권한 프로그램을 제어합니다.

설정은 현재 Windows 사용자 계정에 저장되며 상대 PC로 복사되지 않습니다.

## Swap mode

양쪽 PC에서 **Swap mode**를 켜고 같은 **Control direction**을 선택합니다.

방향은 항상 제어 PC를 기준으로 합니다. 예를 들어 `Left`는 제어 PC의 왼쪽 경계에 포인터를 1초간 두면 대기 PC로 제어가 넘어간다는 뜻입니다. 돌아올 때는 복귀 경계에서 한 번 벗어난 뒤 다시 1초간 머뭅니다.

양쪽 방향이 다르거나 한쪽 PC에서만 Swap mode가 켜져 있으면 Soft KVM이 설정 수정을 안내합니다.

## Game input mode

게임에서 카메라를 계속 움직여야 할 때 **Game input mode**를 사용합니다. 일반 바탕화면이나 절대 좌표 클릭이 필요한 게임 메뉴를 사용할 때는 끕니다.

두 PC에서 같은 최신 Soft KVM EXE를 사용해야 합니다. Windows 합성 입력을 거부하는 안티치트 게임은 지원하지 않습니다.

## 업데이트와 연결 확인

- Settings의 **Check for updates**에서 최신 공개 버전을 설치할 수 있습니다.
- Windows Defender 방화벽이 물으면 사설 네트워크 통신을 허용합니다.
- 두 PC를 같은 신뢰 가능한 로컬 네트워크에 연결합니다.
- 연결되지 않으면 양쪽 버전, Connection group, Swap mode 설정이 같은지 확인합니다.
- Stop, Pause, 연결 종료 또는 긴급 복구를 실행하면 전달 중인 키와 마우스 버튼이 해제됩니다.

> Soft KVM은 신뢰할 수 있는 로컬 네트워크에서만 사용하세요. 입력 트래픽은 암호화되지 않으므로 공용 Wi-Fi나 인터넷에 직접 노출하면 안 됩니다.
